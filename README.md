# Hospital Management System (PostgreSQL)

A real-world Hospital Management System database built using PostgreSQL.

## Features
- Patient Management
- Doctor Management
- Appointment Scheduling
- Diagnosis Tracking
- Prescription Management
- Admission & Room Allocation
- Billing & Payment Processing
- Business Analytics

## Database Tables
1. Departments
2. Doctors
3. Patients
4. Appointments
5. Diagnoses
6. Prescriptions
7. Rooms
8. Admissions
9. Bills
10. Payments
