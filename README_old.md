)# Hospital Management Database

## Project Overview
A PostgreSQL database project designed to manage hospital operations including patients, doctors, appointments, admissions, rooms, billing, and prescriptions.

## Technologies Used
- PostgreSQL
- pgAdmin 4
- SQL
- dbdiagram.io

## Database Tables
- departments
- doctors
- patients
- appointments
- rooms
- admissions
- bills
- prescriptions

## Skills Demonstrated
- Database Design
- ER Modeling
- Primary Keys
- Foreign Keys
- Joins
- Aggregations
- Views
- Stored Procedures
